# Industry Solutions Finder Agent

A specialized agent designed to help users discover relevant Google Cloud solutions for their industry.

This agent is built using the **[Google Agent Development Kit (ADK)](https://google.github.io/adk-docs/)** and leverages a built-in knowledge base of industry-specific solutions.

## Features

-   **Industry Discovery**: Prompts users to identify their industry.
-   **Solution Lookup**: Retrieves key focus areas and details for the selected industry using a dedicated tool.
-   **Guided Interaction**: Ensures the user stays focused on finding industry solutions.

## Usage

This agent can be used to quickly explore Google Cloud's offerings across various sectors such as:
-   Financial Services
-   Healthcare & Life Sciences
-   Retail & CPG
-   Manufacturing & Automotive
-   And more...

## Tools

*   `get_industry_solutions`: A tool that takes an industry name as input and returns the key focus areas and descriptions.
